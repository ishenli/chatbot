## 说明

基于安卓最新的 jetpack 技术栈实现的个人助理应用，开发的语言是 Java。

### 相关技术
+ MVVM
+ Material Design 3
+ LiveData
+ Room
+ Retrofit2

### 准备工作

#### 安装本地 gradle

打开 `gradle/wrapper/gradle-wrapper.properties` 修改 gradle-bin 的地址，本地是

`distributionUrl=file\:/${本地路径}/gradle-8.3-bin.zip`
